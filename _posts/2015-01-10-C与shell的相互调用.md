## Shell脚本调用C语言

直接使用命令启动C语言程序+参数即可。
例如：
{% highlight ruby %}
.bin/main 1 2
{% endhighlight %}

## C程序调用Shell脚本

### 方法一：

{% highlight ruby %}
#include "stdio.h"
void main()
{
   system("ls");
}
{% endhighlight %}

### 方法二：

{% highlight ruby %}
#include "stdio.h"
void main()
{
   FILE * fp; 
   char buffer[80];
   fp=popen("pwd","r");
   fgets(buffer,sizeof(buffer),fp);
   printf("%s",buffer);
   pclose(fp);
}
{% endhighlight %}
